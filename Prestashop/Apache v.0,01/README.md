# Prestashop

### Build dockerfile
  > docker build -t cpa .

### Variables dockerfile

  > ENV A2CONF "nombre archiv .conf del apache"
  
  > ENV DROOT "ruta de directorio root del site apache"

### Lista de paquetes instalados
  * apache2
  * apache2-doc
  * php7.0
  * libapache2-mod-php7.0
  * php7.0-pspell
  * php7.0-curl
  * php7.0-gd
  * php7.0-intl
  * php7.0-mysql
  * php7.0-xmlrpc
  * php7.0-ldap
  * php7.0-xml
  * php7.0-zip
  * unzip
